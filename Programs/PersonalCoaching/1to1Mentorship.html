Syllabus

Duration: 1 Month, 3 Month, 6 Month

Excel: Beginner to Advanced

Excel Data Analysis Data Manipulation
CHAPTER 1 - Kick Start Microsoft Excel

This chapter provides an overview of Excel's interface, essential shortcuts, and basic functionalities to help users navigate and work efficiently.

CHAPTER 2 - Data Handling & Formatting

Learn how to organize, clean, and format data effectively using conditional formatting, data validation, and number formatting techniques.

CHAPTER 3 - Working with Formulas

Master essential Excel formulas, including mathematical operations, logical functions, lookup functions, and advanced calculations for data analysis.

CHAPTER 4 - Text and Date Functions

Learn how to manipulate text and date values in Excel using functions like CONCATENATE, LEFT, RIGHT, MID, TEXT, TODAY, and NETWORKDAYS.

CHAPTER 5 - Data Cleaning & Preparation

Understand techniques for handling missing values, removing duplicates, and using functions like TRIM, SUBSTITUTE, and CLEAN for data preparation.

CHAPTER 6 - Data Analysis using Pivot Tables

Learn to summarize and analyze large datasets efficiently using Pivot Tables, filters, grouping, and Pivot Charts.

CHAPTER 7 - Data Visualization with Charts & Graphs

Understand different types of charts and how to visualize data effectively using bar charts, line charts, pie charts, and dynamic charting techniques.

CHAPTER 8 - Advanced Excel Techniques

Explore Named Ranges, Dynamic Arrays, What-If Analysis, Goal Seek, and Solver to enhance data analysis capabilities.

CHAPTER 9 - Introduction to Macros & Automation

Learn how to automate repetitive tasks using Excel Macros and VBA basics for increased efficiency.

CHAPTER 10 - Case Study & Capstone Project

Apply all the skills learned to real-world scenarios, clean and analyze a dataset, create an automated dashboard, and present insights.

SQL: Beginner to Advanced

SQL Server Data Manipulation Case Study
CHAPTER 1 - Introduction to SQL & Databases

Understanding databases, SQL fundamentals, and basic commands to create and manage tables.

CHAPTER 2 - Querying Data with SELECT & WHERE

Using SELECT, WHERE clause, and filtering techniques to retrieve specific data.

CHAPTER 3 - Data Aggregation with GROUP BY & HAVING

Learning how to summarize and group data using GROUP BY and filter grouped data with HAVING.

CHAPTER 4 - Sorting & Ordering Data

Using ORDER BY to sort data and understanding Query Execution Order.

CHAPTER 5 - Joins & Set Operators

Exploring INNER JOIN, LEFT JOIN, RIGHT JOIN, NATURAL JOIN, UNION, and INTERSECT.

CHAPTER 6 - Advanced Joins & Nested Queries

Understanding SELF JOIN, CROSS JOIN, nested queries, and Common Table Expressions (CTEs).

CHAPTER 7 - Built-in Functions & Conditional Logic

Using string, numeric, and date functions along with CASE WHEN for conditional statements.

CHAPTER 8 - Temporary Tables, Views & Partitioning

Exploring temp tables, views, PIVOT, and partitioning techniques.

CHAPTER 9 - SQL Procedures, Triggers & Performance Tuning

Understanding stored procedures, triggers, and optimizing queries for performance.

CHAPTER 10 - Real-world Case Study & Final Project

Applying SQL skills to analyze data, create reports, and solve a real-world problem.

Python: Beginner to Advanced for Data Analysis

Data Analysis Pandas Data Visualization Automation
CHAPTER 1 - Introduction to Python for Data Analysis

Understanding Python's role in data analysis, setting up the environment, and basic syntax.

CHAPTER 2 - Working with Data Structures

Lists, Tuples, Dictionaries, and their applications in data handling.

CHAPTER 3 - Data Handling with NumPy

Using NumPy for numerical operations, arrays, and performance optimization.

CHAPTER 4 - Data Manipulation with Pandas

Reading, cleaning, and transforming datasets using Pandas DataFrames.

CHAPTER 5 - Data Visualization with Matplotlib & Seaborn

Creating meaningful charts and visual representations of data.

CHAPTER 6 - Exploratory Data Analysis (EDA)

Techniques to understand data distribution, patterns, and outliers.

CHAPTER 7 - Working with Databases using SQL in Python

Connecting to databases, running SQL queries with Python using SQLite and Pandas.

CHAPTER 8 - Data Automation and Scripting

Automating repetitive tasks like data cleaning and report generation using Python.

CHAPTER 9 - Introduction to Statistics for Data Analysis

Basic statistical concepts, measures of central tendency, and correlation.

CHAPTER 10 - Case Study & Final Project

Applying Python skills to analyze real-world datasets and present insights.

Power BI: Beginner to Advanced

Power Query Data Modeling DAX Dashboards
CHAPTER 1 - Introduction to Power BI

Overview of Power BI, installation, interface navigation, and basic workflow.

CHAPTER 2 - Connecting to Data Sources

Importing data from Excel, databases, web, and APIs into Power BI.

CHAPTER 3 - Data Transformation with Power Query

Cleaning, shaping, and transforming raw data using Power Query Editor.

CHAPTER 4 - Advanced Power Query Techniques

Merging, appending, filtering, pivoting, and creating custom columns.

CHAPTER 5 - Introduction to Data Modeling

Understanding relationships, cardinality, and star schema modeling.

CHAPTER 6 - Advanced Data Modeling

Creating calculated tables, hierarchies, and optimizing data models.

CHAPTER 7 - Introduction to DAX (Data Analysis Expressions)

Fundamentals of DAX, measures vs calculated columns, and basic functions.

CHAPTER 8 - Advanced DAX Formulas

Using FILTER, SUMX, RANKX, and context transition for deep analysis.

CHAPTER 9 - Time Intelligence in DAX

Implementing YTD, MTD, QTD, running totals, and other time-based calculations.

CHAPTER 10 - Creating Interactive Visualizations

Building bar charts, line charts, slicers, maps, and KPI indicators.

CHAPTER 11 - Advanced Visualizations & Custom Charts

Working with custom visuals, conditional formatting, and storytelling techniques.

CHAPTER 12 - Building Dashboards & Reports

Designing professional dashboards, page navigation, and user experience best practices.

CHAPTER 13 - Publishing & Sharing Reports

Publishing reports to Power BI Service, setting up workspaces, and sharing dashboards.

CHAPTER 14 - Power BI Automation & Performance Optimization

Optimizing queries, incremental refresh, and automating data updates.

CHAPTER 15 - Real-world Case Study & Final Project

Applying Power BI skills to build a complete data-driven dashboard and business report.

Business Acumen: High-Demand Guide

Guestimates Case Study Domain KT
CHAPTER 1 - Introduction to Business Acumen

Understanding business fundamentals, key business drivers, and decision-making processes.

CHAPTER 2 - Understanding Industry & Market Trends

Exploring industry structures, competitive landscapes, and market dynamics.

CHAPTER 3 - Problem-Solving with Guestimates

Learning guestimation techniques for business scenarios and case interviews.

CHAPTER 4 - Case Study: Business Growth Strategies

Analyzing real-world business growth strategies using structured problem-solving.

CHAPTER 5 - Financial Acumen for Business Decision Making

Interpreting financial statements, budgeting, and profitability analysis.

CHAPTER 6 - Domain Knowledge Transfer (KT) Essentials

Learning about major industries: Retail, Finance, Healthcare, and IT.

CHAPTER 7 - Competitive Analysis & Business Strategy

Understanding Porter’s Five Forces, SWOT analysis, and strategic frameworks.

CHAPTER 8 - KPIs & Metrics in Business Analysis

Identifying and tracking key performance indicators across different industries.

CHAPTER 9 - Business Forecasting & Decision Making

Understanding demand forecasting, data-driven decision making, and risk management.

CHAPTER 10 - Case Study: Digital Transformation & Business Innovation

Exploring how businesses leverage technology to drive innovation and efficiency.
